  • Has over 25,000 Contributors
  • Is a Stable, Secure, Powerful and User-Friendly OS
  • Is Upstream for RHEL, OLPC and Others
  • Has Over 10,000 Packages
  • Based on Yum package manager




  • Dedication to free software and content
  • Releases that are predictable and 100% legally redistributable for everyone
  • Completely free project that anyone can emulate or copy in whole or in part for their own purposes

FEDORA TOOLS


  • Publican (Documentation)
  • ABRT
  • Developer-Assistant feature (F19)

Publican


  • Tool for publishing material authored in DocBook XML in pdf
  • Ensures validity of DocBook XML markup
  • Supports All Languages
  • Integrates well with versioning systems


   Major features


  • create_book (XML -> HTML)
  • make dist-srpm (Tarball - src.rpm)

ABRT


  • ABRT, Apport, Bug-buddy, Anaconda, Kernel-oops
  • Can intercept crashes immediately
  • Can gather potentially useful information about the crash and the OS
  • Can be auto-invoked for auto-detectable problems
  • User-Friendly UI for helping users file bugs on crashes

SELINUX


  • Developed by National Security Agency (NSA)
  • Processes and files labelled with a type
  • Not a Linux distribution, but kernel modifications
  • Processes run in their own domain
  • SELinux policy rules define how processes interact with each other
  • Default setting is  "NO ACCESS"
  • SELinux adds Mandatory Access Control to the Kernel



Fedora spins


  • Spins are official and carry Fedora Branding
  • AOS Spin - Meant for running applications
  • BrOffice.org Spin
  • Education Spin
  • Electronic Lab Spin: FEL is Fedora's high-end hardware design and simulation platform. Includes EDA Package ( Exploratory data analysis )
  • Games, Geo spins
  • Spins need validation before submission for approval
